# Kuran Nüzul Sırası Mobil & Web Uygulaması

Kullanıcıların Kuran-ı Kerim'i nüzul sırasına göre okumasını sağlayan kullanıcı dostu Flutter arayüzüdür.

## ✨ Kullanıcı Deneyimi Yetenekleri
- **Responsive Tasarım:** Mobil (Android/iOS) ve Web tarayıcıları için optimize edilmiş tek kod tabanı.
- **Nüzul Sıralı Liste:** Sureleri klasik mushaf sırası yerine indirilme tarihine göre keşfetme imkanı.
- **Detaylı Okuma:** Sure detaylarında ayetlerin Arapça metni, Türkçe meali ve AI özetlerini görebilme.
- **Kullanıcı Girişi:** "test/test" bilgileriyle giriş yaparak okuma ilerlemesini bulutta saklama.
- **Misafir Modu:** Giriş yapmadan tüm sure ve ayet içeriğine tam erişim.
- **Modern UI:** Temiz, hızlı ve odaklanmayı kolaylaştıran liste görünümü.

## 🛠️ Teknik Özellikler
- **State Management:** Provider ile hızlı ve kararlı durum yönetimi.
- **Networking:** Dio kütüphanesi ile verimli API iletişimi.
- **Modüler Yapı:** Kolayca genişletilebilir model-servis-provider mimarisi.

## 📱 Platform Desteği
- Web (HTML/CanvasKit)
- Android (APK/App Bundle)
- iOS (IPA)
